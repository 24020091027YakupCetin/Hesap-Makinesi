# Hesap Makinesi

Bu proje, Java programlama dili kullanılarak geliştirilmiş basit bir hesap makinesi uygulamasıdır. 
Program, kullanıcıdan alınan iki sayı üzerinde temel matematiksel işlemleri gerçekleştirmektedir.

## Proje İçeriği

Uygulama aşağıdaki işlemleri destekler:

- Toplama (+)
- Çıkarma (-)
- Çarpma (*)
- Bölme (/)

Kullanıcı konsol üzerinden iki sayı girer ve yapmak istediği işlemi seçer. 
Program, seçilen işleme göre sonucu hesaplayarak ekrana yazdırır.

## Kullanılan Teknoloji

- Java
